I make web things.

Mobile-first

Websites are no longer a desktop affair. Modern design requires a mobile-first mentality, to ensure the maximum amount of users can utilize that beautiful code you're producing.

Modern Tech

New tech is being developed daily. Innovative design should be leveraged anytime possible, and unless a developer stays on top of current trends, he could get left behind.

Fast

I set out to build lean, fast applications. People could be viewing this on the latest Alienware gaming system with the most up to date browsers, or on a backwoods Compaq with Windows XP. Thestuff I build should run smoothly no matter the platform.
